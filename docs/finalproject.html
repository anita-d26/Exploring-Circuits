FlashMemory!

Concept

FlashMemory is an interactive LED-based memory game that helps users improve short-term memory and pattern recognition skills.

It responds to a start button press by generating a blinking LED sequence, and it provides visual feedback when it detects correct or incorrect user input.

Input: Push button (start/reset), keyboard input via Serial.
Output: LED blinking sequences, success/failure feedback patterns, and score tracking via Serial Monitor.

Anticipated Bill of Materials

  • Arduino Uno R3
  • Breadboard
  • 4 LEDs (red, blue, yellow, green)
  • Push button (start/reset)
  • 220Ω resistors (for LEDs)
  • 10kΩ resistor (pull-down resistor)
  • Jumper wires
  • USB cable
  • Computer with Arduino IDE (uses digital logic)
  • Enclosure (final prototype housing)

Timeline and Contingency Plans

  • Week 1: Build basic LED + button circuit and program blinking sequence.
    • If encountering circuit issues -> test one LED first to verify correct behavior before incorportating all 4.
  • Week 2 - Implement random sequence generator, feedback logic, and level progression.
    • If random sequence generation does not work as planned -> implement a fixed hardcoded sequence to demonstrate functionality.
    • If level progression logic too complex -> simplify to a basic incrementing difficulty level.
  • Week 3: Finalize enclosure, clean wiring, and complete full system testing.
    • If enclosure fabrication becomes time-constrained -> create a cardboard housing to ensure the project is integrated into a physical product.
